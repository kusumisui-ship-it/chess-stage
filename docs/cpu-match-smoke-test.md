# CPU MATCH — Smoke Test Checklist

CPU戦の基本動作を、スマートフォン実機で短時間に確認するためのチェックリスト。

## 1. 起動

- [ ] HOMEから「対局」を開ける
- [ ] 「CPU戦」を押すと盤面へ遷移する
- [ ] 初期配置が正しい
- [ ] 白番が操作可能になっている
- [ ] `WHITE TO MOVE` と表示される

## 2. 基本操作

- [ ] 白の駒を押すと選択状態が表示される
- [ ] 合法手のみ表示される
- [ ] 自駒のあるマスへは移動できない
- [ ] 違う白駒を押すと選択対象が切り替わる
- [ ] 空きマスを押すと選択が解除される

## 3. CPU応手

- [ ] 白が指した後に `BLACK THINKING` と表示される
- [ ] CPUが自動で1手指す
- [ ] CPUの手番中は白の駒を操作できない
- [ ] CPU応手後に白番へ戻る
- [ ] 10手以上続けてもCPUが停止しない

## 4. 特殊ルール

- [ ] キャスリングできる条件で実行できる
- [ ] チェック中はキャスリングできない
- [ ] 通過マスが攻撃されている場合はキャスリングできない
- [ ] アンパッサンが成立する
- [ ] ポーン昇格時に選択画面が出る
- [ ] クイーン、ルーク、ビショップ、ナイトを選べる

## 5. 勝敗

- [ ] チェック中のキングが赤く表示される
- [ ] チェックメイトで対局が終了する
- [ ] ステイルメイトで引き分けになる
- [ ] RESIGNで黒勝ちになる
- [ ] 終局後は盤面を操作できない

## 6. 画面遷移と連打耐性

- [ ] CPU思考中にHOMEへ戻っても後からCPUが動かない
- [ ] CPU思考中にRESIGNしても後からCPUが動かない
- [ ] REMATCHで新しい初期局面になる
- [ ] REMATCHを連打しても壊れない
- [ ] 盤面反転を複数回行っても操作位置がずれない
- [ ] ASSISTのON/OFFが正しく反映される

## 不具合記録テンプレート

```text
端末:
ブラウザ:
発生した手順:
1.
2.
3.

期待した動作:
実際の動作:
再現率: /5回
スクリーンショット:
```
