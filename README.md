# CHESS STAGE

「チェスのルールは変えない。体験を変える。」

現在の `main` は、スマートフォン優先のオフラインテストビルドです。

## 実装済み

- CPU戦（EASY / NORMAL / HARD）
- 同じ端末での2人対局
- 白・黒・ランダム選択
- 無制限 / 5分 / 10分 / 15分
- キャスリング、アンパッサン、プロモーション
- チェック、チェックメイト、ステイルメイト
- デッドポジション、3回同一局面・50手ルールの請求、5回同一局面・75手ルール
- 合法手、直前の手、座標表示
- UNDO、盤面反転、投了、引き分け請求
- SAN棋譜、PGNコピー、FEN開始
- 対局の自動保存・再開
- ローカル棋譜アーカイブと戦績
- 特殊ルールと1手詰めのミニレッスン

## 今回の対象外

- オンライン対戦
- 観戦共有
- アカウント・同期
- 課金・ショップ・スキン購入
- サーバー側モデレーション

## 構成

- `engine-core.js` — 盤面・駒・特殊手の基礎処理
- `engine-rules.js` — 合法手、勝敗、SAN / PGN、UNDO
- `engine-ai.js` — CPU評価と探索
- `app-core.js` — 状態・保存・共通処理
- `app-views.js` — ホーム、記録、学習、設定
- `app-game.js` — 対局画面と対局進行
- `app-runtime.js` — イベント、時計、保存処理

## エンジンテスト

```bash
node tests/engine.test.js
```
